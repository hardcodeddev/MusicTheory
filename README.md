# Dubstep Theory Quest

A browser-based learning game that takes you from zero music knowledge to dubstep-ready theory and chord skills.

## Features
- Structured levels that teach rhythm, scales, chords, progressions, and bassline writing.
- Chord builder with dubstep-friendly flavors (minor 7, sus chords, dominant 7) and onboard audio playback.
- Ear-training quiz that plays random chords so you can identify qualities and tension.
- Local progress tracking with XP; reset anytime.

## Usage
Open `index.html` in your browser. Progress is stored in localStorage.
